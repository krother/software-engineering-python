
# Minesweeper

Schreibe ein Programm, das ein Minesweeper-Feld mit der Anzahl benachbarter Minen befüllt. Aus:

    .....*..
    .*......
    ......*.
    ..**....
    ..*...*.
    .......*

wird so:

    11101*10
    1*100111
    023211*1
    02**1222
    02*311*2
    0111012*

Wähle eine geeignete Datenstruktur aus. Schreibe einen Test für das Programm, bevor Du es implementierst.
